# Tic-Tac-Toe-Game
A GUI-based Tic Tac Toe game built with Python and Tkinter featuring Player vs Player and Player vs Computer modes.

## Features

- Play against the computer
- Two-player mode
- Choose X or O
- Select who starts first
- Detect wins and draws
- Highlight the winning combination
- Restart the game after each round

## Technologies

- Python
- Tkinter
- Event-driven Programming

## AI Strategy

The computer player:

- Tries to win whenever possible
- Blocks the opponent from winning
- Prioritizes the center square
- Chooses corners before edges

## How to Run

```bash
python tic_tac_toe.py
```
